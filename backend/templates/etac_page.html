{% trans "— FAQ" %}

{% trans "よくあるご質問" %}

{% trans "ご検討中のお客様からよくいただく質問をまとめました。" %}

{% trans "アウトソーシングとは具体的に何ですか？" %}
+

{% trans "アウトソーシングとは、自社内で行っていた業務を外部の専門企業に委託することです。採用・育成・製造・事務などのノンコア業務を外部に任せることで、貴社のコア業務に集中できるようになります。" %}

{% trans "最低何名から依頼できますか？" %}
+

{% trans "1名からでも対応可能です。ただし、コスト効率の観点から3名以上のご依頼がより大きなメリットを感じていただけます。まずはご要件をお聞かせください。" %}

{% trans "相談から実際の業務開始まで、どれくらいかかりますか？" %}
+

{% trans "ヒアリング（1〜2週間）→ 調査・分析（1〜2週間）→ 契約・体制構築（約45日）→ トライアル運用（3〜4週間）の流れで、最短約3ヶ月での本格稼働が可能です。" %}

{% trans "機密情報や社内データの管理は安全ですか？" %}
+

{% trans "情報管理には細心の注意を払っています。委託開始前に必ず秘密保持契約（NDA）を締結します。IT業務については現地コーディネート企業との共同管理体制を構築しており、情報漏洩リスクを最小限に抑えています。" %}

{% trans "外国人材のビザや生活面のサポートも含まれますか？" %}
+

{% trans "はい、すべて含まれます。登録支援機関として、ビザ申請・在留資格管理・住居手配・生活支援・日本語研修まで一括でサポートします。クライアント様が手続きに関わる必要は一切ありません。" %}

{% trans "スタッフのパフォーマンスが期待を下回った場合はどうなりますか？" %}
+

{% trans "弊社が責任をもって代替スタッフの手配または追加研修・指導を行います。定期的なパフォーマンス評価と報告を実施しており、問題の早期発見・対応に努めています。スタッフの定着率90%以上という実績がその証明です。" %}

{% trans "繁忙期・閑散期に応じて人数を増減できますか？" %}
+

{% trans "はい、これがアウトソーシングの大きなメリットの一つです。正社員と違い、業務量に応じて人員を柔軟に調整できます。繁忙期に増員、閑散期に縮小するなど柔軟な対応が可能です。" %}

{% trans "費用はどのように決まりますか？" %}
+

{% trans "業務内容・人員数・スキルレベル・対応期間などをもとに見積もりをご提示します。ヒアリング後、迅速に概算をお出しします。固定費を変動費に転換できるため、多くの場合コスト削減につながります。" %}

{% trans "製造業以外の業種でも対応できますか？" %}
+

{% trans "はい。製造業を主力としていますが、IT・介護・建設・飲食・農業・物流・ホテルなど多様な業種に対応しています。e-tacグループ4社の連携により、各業種に特化した専門チームでサポートします。" %}
